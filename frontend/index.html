How to read a baseline profile
Usual hours show when this IP normally appears. Off-hours activity can mean attacker activity or abused automation.
Usual ports show expected services. A new SSH/RDP/database port is more suspicious than a repeated normal web port.
Deviation alerts are the concrete reasons this IP broke its own behavioural baseline.

How to read ML anomaly scoring

          

            
Anomaly score is the raw Isolation Forest output. Lower/negative means the IP looks
              more unusual compared with other IPs.

            
Risk score is the SOC-facing 0-100 danger score. It combines ML abnormality plus
              baseline and threat signals.

            
Confidence says how much evidence supports that risk. Few logs means lower
              confidence, even if the IP looks suspicious.

            
Isolation Forest finds IPs that are isolated from normal behaviour patterns. It
              does not know "bad"; it finds outliers.
